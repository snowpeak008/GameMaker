# NEWrust Gates

This directory stores local gate evidence generated by `adm-new-cli`.

- `plan-gate.adm`: plan and scorecard gate output.
- `parity-gate.adm`: Rust parity gate output.
- `ui-gate.adm`: Web e2e and screenshot gate output.
- `package-gate.adm`: package success/blocked evidence output.
- `release-gate.adm`: aggregated release gate output.
- `handoff-gate.adm`: final handoff command output.
- `reports/final_handoff_manifest.json`: machine-readable Python evidence to NEWrust implementation map.
- `reports/final_handoff_manifest.md`: human-readable handoff summary.

Gate reports are generated artifacts. Regenerate them with the matching CLI command instead of editing them by hand.
