# Session 2026-07-10-002

## Summary

Added replaceable pure Simplified Chinese and pure English application-language modes
to the NEWrust/Tauri product without adding a visible selector. The desktop captures
`ADM_NEWRUST_LANGUAGE` at startup, the Web UI renders from one symmetric localization
catalog, the built-in design taxonomy is overlaid by stable ID, and future settings UI
can reuse the existing persisted replacement API.
