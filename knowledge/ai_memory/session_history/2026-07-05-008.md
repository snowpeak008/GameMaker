## 完成内容

- `adm-ai`
  - 新增稳定字符串表示与解析：
    - `AiCapability::as_str/parse`
    - `AiOutputState::as_str/parse`
    - `AiTaskStatus::as_str/parse`
  - `AiTaskJournal::render()` 改为 `ADM_AI_JOURNAL_V1` 格式。
  - 新增 `AiTaskJournal::from_text(...)`。
  - 新增 `AiTaskJournal::save_to_path(...)`。
  - 新增 `AiTaskJournal::load_from_path(...)`。
  - journal 现在可恢复：
    - task id
    - capability
    - prompt
    - context summary
    - provider id
    - status
    - budget units
    - attempts
    - created/updated timestamps
    - error
    - result task id
    - result provider id
    - output state
    - raw output
    - validation notes
  - prompt、context、raw output、error 和 validation notes 使用十六进制编码，避免多行文本或 `=` 破坏解析。
- `adm-application`
  - 核心流水线测试新增从正式存档 `content/ai/journal.adm` 反解析 journal 的断言。
  - 应用层继续写入同一路径，但内容已经是 V1 可恢复格式。
