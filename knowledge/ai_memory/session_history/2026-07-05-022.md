# 会话记录：2026-07-05-022

## 摘要

继续 goal 模式推进 Rust 全项目重建，在 `adm-ai` 中增加 chat-completions 兼容 JSON transport。该 transport 通过注入的 `AiHttpJsonClient` 发送 JSON，不直接绑定真实网络库；测试使用 fake HTTP client，避免真实网络依赖。

## 完成内容

- `adm-ai`
  - `Cargo.toml` 新增 `serde_json` 直接依赖。
  - 新增 `AiHttpJsonClient` trait。
    - 方法：`post_json(endpoint, bearer_secret, request_body)`。
    - 后续可由 reqwest/Tauri/平台 HTTP 客户端实现。
  - 新增 `ChatCompletionsTransport<C: AiHttpJsonClient>`。
    - 校验 model 非空。
    - 将 `AiRemoteRequest` 转换为 chat-completions 风格 JSON。
    - endpoint hint 若未以 `/chat/completions` 结尾，则自动补齐。
    - 将 runtime secret 仅作为 bearer secret 传给 HTTP client。
    - 解析 `choices[0].message.content`。
    - 缺失或空 content 返回 validation error。

    - 验证 response content 解析。

    - 验证缺少 message content 时失败。
