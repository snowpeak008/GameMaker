# A02 验证与哈希

状态：待执行
依赖：A01

## 目标

让 `GameSpec` 在进入任何编译阶段前可确定性判定有效性和身份。

## 变更

- 检查 ID 唯一性、引用存在性、状态迁移闭合、动作条件/效果、资源约束、场景引用和追踪闭合。
- 建立稳定排序、规范 JSON 和 SHA-256 内容哈希。
- 错误包含稳定代码、严重级别、规范路径、相关 ID 和修复提示。
- 添加 mutation tests：删除引用、重复 ID、无效迁移、空效果、断裂追踪必须失败。

## 验收

同一语义输入在字段/集合合法重排后得到同一规范哈希；所有无效样例 fail closed。
